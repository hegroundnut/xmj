## 📋 分层架构说明

### Controller层 (控制器)
**职责**:
- 处理HTTP请求
- 参数验证和获取
- 调用Service层
- 统一返回响应格式

**规范要求**:
```php
<?php
namespace app\adminapi\controller\v1\user;

use app\Request;
use app\services\user\UserServices;
use crmeb\basic\BaseController;

/**
 * 用户管理控制器
 */
class UserController extends BaseController
{
    protected $services;

    public function __construct(UserServices $services)
    {
        $this->services = $services;
    }

    /**
     * 获取用户列表
     */
    public function getList()
    {
        $where = $this->request->getMore([
            ['keywords', ''],
            ['status', ''],
            ['page', 1],
            ['limit', 15]
        ]);
        $list = $this->services->getUserList($where);
        return app('json')->success($list);
    }

    /**
     * 保存用户信息
     */
    public function save()
    {
        $data = $this->request->postMore([
            ['nickname', ''],
            ['phone', ''],
            ['avatar', ''],
            ['status', 1]
        ]);
        $this->services->saveUser($data);
        return app('json')->success('保存成功');
    }
}
```

**命名规范**:
- 控制器方法: 小写+下划线,如 `get_list`, `save_data`
- 参数获取: 使用 `$this->request->getMore()` 或 `$this->request->postMore()`
- 返回响应: `return app('json')->success($data)` 或 `app('json')->fail('错误信息')`

### Service层 (服务层)
**职责**:
- 实现核心业务逻辑
- 事务管理
- 调用DAO层
- 缓存处理

**规范要求**:
```php
<?php
namespace app\services\user;

use crmeb\basic\BaseServices;
use app\dao\user\UserDao;
use crmeb\exceptions\AdminException;

class UserServices extends BaseServices
{
    protected $dao;

    public function __construct(UserDao $dao)
    {
        $this->dao = $dao;
    }

    /**
     * 获取用户列表
     * @param array $where
     * @return array
     */
    public function getUserList(array $where): array
    {
        $query = $this->dao->search($where);

        // 获取总数
        $count = $query->count();

        // 分页获取数据
        $list = $query->page($where['page'], $where['limit'])
            ->order('id', 'desc')
            ->select()
            ->toArray();

        // 处理数据
        foreach ($list as &$item) {
            $item['status_text'] = $item['status'] == 1 ? '启用' : '禁用';
        }

        return compact('list', 'count');
    }

    /**
     * 保存用户信息
     * @param array $data
     * @return int
     * @throws AdminException
     */
    public function saveUser(array $data): int
    {
        // 参数验证
        if (empty($data['phone'])) {
            throw new AdminException('手机号不能为空');
        }

        // 检查手机号是否已存在
        if (isset($data['id'])) {
            $count = $this->dao->where('phone', $data['phone'])
                ->where('id', '<>', $data['id'])
                ->count();
            if ($count > 0) {
                throw new AdminException('手机号已存在');
            }
        }

        // 保存数据
        if (isset($data['id']) && $data['id']) {
            $id = $data['id'];
            unset($data['id']);
            $this->dao->update($id, $data);
        } else {
            $id = $this->dao->save($data);
        }

        return $id;
    }
}
```

**命名规范**:
- 服务方法: 驼峰法,如 `getUserList`, `saveUser`
- 参数类型声明: 必须声明参数和返回值类型
- 异常处理: 使用项目异常类,如 `AdminException`, `ApiException`

### DAO层 (数据访问层)
**职责**:
- 封装数据库操作
- 提供数据查询方法
- 处理复杂的SQL查询

**规范要求**:
```php
<?php
namespace app\dao\user;

use crmeb\basic\BaseDao;
use app\model\user\User;

class UserDao extends BaseDao
{
    /**
     * 设置模型
     * @return string
     */
    protected function setModel(): string
    {
        return User::class;
    }

    /**
     * 搜索条件
     * @param array $where
     * @return \think\Model
     */
    public function search(array $where = [])
    {
        $query = $this->getModel()->where('is_del', 0);

        // 关键词搜索
        if (!empty($where['keywords'])) {
            $query = $query->whereLike('nickname|phone', "%{$where['keywords']}%");
        }

        // 状态筛选
        if ($where['status'] !== '') {
            $query = $query->where('status', $where['status']);
        }

        return $query;
    }

    /**
     * 根据ID获取用户信息
     * @param int $id
     * @return array|null
     */
    public function getById(int $id): ?array
    {
        return $this->getModel()->where('id', $id)->find();
    }
}
```

**命名规范**:
- DAO方法: 数据库操作相关,如 `search`, `getById`, `update`, `delete`
- 查询构造: 必须返回 `\think\Model` 对象
- 模型绑定: 必须实现 `setModel()` 方法

### Model层 (模型层)
**职责**:
- 定义数据模型
- 设置表名和主键
- 定义关联关系
- 定义访问器和修改器

**规范要求**:
```php
<?php
namespace app\model\user;

use crmeb\basic\BaseModel;
use think\model\relation\HasMany;

class User extends BaseModel
{
    protected $name = 'user';
    protected $pk = 'id';

    /**
     * 关联订单
     * @return HasMany
     */
    public function orders()
    {
        return $this->hasMany(Order::class, 'uid', 'id');
    }

    /**
     * 头像获取器
     * @param $value
     * @return string
     */
    public function getAvatarAttr($value): string
    {
        return $value ? $value : '/static/default_avatar.png';
    }

    /**
     * 状态文本获取器
     * @param $value
     * @return string
     */
    public function getStatusTextAttr($value, $data): string
    {
        $status = [0 => '禁用', 1 => '启用'];
        return $status[$data['status']] ?? '';
    }

    /**
     * 自动时间戳
     * @var bool
     */
    protected $autoWriteTimestamp = true;
}
```

#### Step 4: 创建Service
```php
// app/services/ExampleServices.php
<?php
namespace app\services;

use crmeb\basic\BaseServices;
use app\dao\ExampleDao;

class ExampleServices extends BaseServices
{
    public function __construct(ExampleDao $dao)
    {
        $this->dao = $dao;
    }

    public function getList(array $where): array
    {
        $query = $this->dao->search($where);
        $count = $query->count();
        $list = $query->page($where['page'], $where['limit'])
            ->order('id', 'desc')
            ->select()
            ->toArray();
        return compact('list', 'count');
    }

    public function save(array $data): int
    {
        if (isset($data['id']) && $data['id']) {
            $id = $data['id'];
            unset($data['id']);
            $this->dao->update($id, $data);
        } else {
            $id = $this->dao->save($data);
        }
        return $id;
    }

    public function delete(int $id): bool
    {
        return $this->dao->update($id, ['is_del' => 1]);
    }
}
```

## 🚀 性能优化建议

### 1. 数据库优化
- 为常用查询字段添加索引
- 避免使用 `SELECT *`,只查询需要的字段
- 使用 `EXPLAIN` 分析SQL执行计划
- 合理使用缓存减少数据库查询

### 2. 缓存优化
- 使用Redis缓存热点数据
- 设置合理的缓存过期时间
- 使用缓存前缀防止冲突
- 缓存更新时注意一致性

### 3. 代码优化
- 减少循环嵌套
- 优化算法复杂度
- 使用队列处理耗时操作
- 异步处理非关键业务

### 4. 接口优化
- 合理使用分页
- 控制返回数据量
- 使用HTTP压缩
- 启用CDN加速
