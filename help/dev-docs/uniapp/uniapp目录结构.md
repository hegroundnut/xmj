### 2.2 目录功能说明

#### 2.2.1 核心目录
- **api/**: 存放 API 接口定义，包含所有后端接口调用
- **components/**: 存放公共组件，可在多个页面复用
- **pages/**: 存放应用页面，每个页面对应一个子目录
- **static/**: 存放静态资源，如图片、字体、图标等
- **utils/**: 存放工具函数，如日期处理、请求封装等

#### 2.2.2 配置目录
- **config/**: 存放应用配置，如 API 地址、常量定义等
- **mixins/**: 存放混入文件，用于复用组件逻辑
- **store/**: 存放 Vuex 状态管理文件
- **libs/**: 存放第三方库，如 SDK、工具库等

#### 2.2.3 插件目录
- **plugin/**: 存放应用插件，如支付插件、分享插件等
