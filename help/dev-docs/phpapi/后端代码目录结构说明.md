# CRMEB 后端代码目录结构说明

## 项目概述

CRMEB 是一个基于 ThinkPHP 6.x 框架开发的电商系统，采用多应用架构模式，支持管理后台、用户端API、客服系统等多个应用模块。

## 核心架构

### 架构模式
- **框架**: ThinkPHP 6.x
- **架构模式**: 多应用架构 (Multi-App)
- **设计模式**: MVC + Service + DAO 分层架构
- **依赖注入**: 容器管理
- **数据库**: MySQL (支持主从分离)
