# CRMEB 项目架构说明

## 🏗️ 整体架构概览

### 技术栈
```
前端：Vue.js 2 + ElementUI + UniApp
后端：ThinkPHP 6.0 + 多应用模式
数据库：MySQL 5.7+
缓存：Redis
消息队列：Workerman
```

### 架构模式
- **MVC 模式**：Model-View-Controller 分层架构
- **三层架构**：Controller → Service → DAO/Model
- **前后端分离**：API 接口与前端页面独立部署
- **多端适配**：支持小程序、H5、APP、PC 端
