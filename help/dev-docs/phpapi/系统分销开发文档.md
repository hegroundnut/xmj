# 系统分销开发文档

## 1. 分销机制概述

CRMEB系统实现了完整的分销功能，支持二级分销、分销等级、佣金计算、分销申请审核等核心功能。分销系统通过推广关系链实现用户之间的推广和佣金分配，帮助商家扩大销售渠道，提高用户粘性。

### 1.1 核心功能

- **二级分销**：支持一级和二级分销关系，可配置分销层级
- **分销等级**：支持多级分销等级，不同等级享受不同的佣金比例
- **佣金计算**：支持按商品售价比例计算佣金，可配置一二级佣金比例
- **自购返佣**：支持用户自购返佣功能，可配置是否开启
- **分销申请**：支持用户申请成为分销员，需管理员审核
- **分销关系管理**：支持查看和管理用户的分销关系链
- **佣金提现**：支持分销员申请提现佣金，管理员审核后发放

        }
        
        // 计算二级佣金

        $cart['two_brokerage'] = $twoBrokerage;

## 10. 最佳实践

### 10.1 分销等级设计

- **合理设置等级数量**：建议设置3-5个分销等级，过多等级会增加管理复杂度
- **梯度设置佣金比例**：不同等级的佣金比例应该有明显差异，激励用户升级
- **设置合理的升级条件**：升级条件应该具有可实现性，避免设置过高的门槛

### 10.2 佣金设置

- **合理设置佣金比例**：根据商品利润率设置合理的佣金比例，避免影响商家利润
- **控制分销层级**：建议使用二级分销，符合国家相关法规要求
- **开启自购返佣**：可以提高用户的购买积极性，增加销售额

### 10.3 推广关系管理

- **明确推广规则**：制定清晰的推广规则，避免用户之间的纠纷
- **定期清理无效关系**：定期清理长期不活跃的推广关系，保持数据整洁
- **保护用户隐私**：在展示推广关系时，注意保护用户的隐私信息
