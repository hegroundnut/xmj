# 业务流程说明

## 📋 核心业务流程

### 🛒 订单创建流程
1. 用户选择商品 → 加入购物车 → 生成订单
2. 订单验证（库存、价格、优惠券）
3. 创建订单数据
4. 扣减库存
5. 发送订单通知

### 💳 支付处理流程  
1. 用户选择支付方式
2. 调用支付接口
3. 支付结果回调
4. 更新订单状态
5. 记录支付流水

### 📦 发货流程
1. 订单状态检查（已支付）
2. 生成发货单
3. 调用物流接口
4. 更新发货状态
5. 通知用户物流信息

### 🔄 售后流程
1. 用户提交售后申请
2. 审核售后请求
3. 处理退款/换货
4. 更新库存和财务
5. 完成售后流程

## ⏱️ 异步任务流程
- 消息队列处理订单相关任务
- 定时任务处理数据统计
- 后台任务处理批量操作
