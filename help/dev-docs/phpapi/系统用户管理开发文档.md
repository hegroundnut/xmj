## 2. 核心功能模块

### 2.1 用户基本管理
- **用户列表**：展示用户基本信息，支持多条件搜索、分页
- **用户详情**：查看用户完整信息，包括基本信息、财务信息、等级信息等
- **用户添加**：后台手动添加用户
- **用户编辑**：修改用户信息
- **用户删除**：删除用户（物理删除/逻辑删除）
- **用户状态管理**：启用/禁用用户账号
- **密码重置**：重置用户登录密码

### 2.2 用户财务与积分管理
- **余额管理**：增加/减少用户余额，生成余额变动记录
- **积分管理**：增加/减少用户积分，生成积分变动记录
- **佣金管理**：管理用户分销佣金，生成佣金记录

### 2.3 会员等级管理
- **等级赠送**：后台手动赠送会员等级
- **等级清除**：清除用户会员等级
- **时长管理**：管理付费会员有效期
- **等级任务**：配置会员等级晋升任务

### 2.4 用户分类管理
- **用户分组**：创建、编辑、删除用户分组，将用户分配到不同分组
- **用户标签**：创建、编辑、删除用户标签，为用户添加/移除标签
- **标签分类**：管理标签分类

### 2.5 推广关系管理
- **推广人管理**：管理用户推广关系
- **推广绑定**：绑定/解绑用户推广关系
- **推广统计**：统计用户推广数据

### 2.6 其他功能
- **微信用户同步**：同步微信用户信息
- **用户注销管理**：处理用户注销请求
- **新人礼配置**：配置新用户注册奖励

系统采用分层架构设计，严格遵循MVC设计模式：
```
├── Model：模型层，定义数据结构和关联关系
├── Dao：数据访问层，处理数据库操作
├── Services：业务逻辑层，实现核心业务逻辑
├── Controller：控制层，处理HTTP请求和响应
└── Route：路由层，定义API路由
```

## 6. 常见问题与解决方案

### 6.1 用户密码加密
**问题**：用户密码如何加密存储？
**解决方案**：系统采用bcrypt算法对用户密码进行加密存储，确保密码安全。

### 6.2 用户登录状态管理
**问题**：如何管理用户登录状态？
**解决方案**：系统使用token机制管理用户登录状态，登录成功后生成token，后续请求携带token进行身份验证。

### 6.3 推广关系绑定
**问题**：如何处理用户推广关系的绑定？
**解决方案**：用户注册时，通过URL参数或邀请码获取推广人ID，自动绑定推广关系。

### 6.4 会员等级有效期计算
**问题**：如何计算会员等级有效期？
**解决方案**：系统根据当前时间加上有效期天数计算会员等级过期时间，过期后自动降级。
