# CRMEB 系统安装流程文档

## 📋 文档概述

本文档提供CRMEB开源商城系统的完整安装流程，适用于CRMEB 5.6.4+版本。

## 🖥️ 服务器环境要求

### 基础配置要求

| 配置项 | 最低配置 | 推荐配置 |
|--------|---------|---------|
| CPU | 1核 | 2核及以上 |
| 内存 | 1GB | 2GB及以上 |
| 硬盘 | 20GB SSD | 50GB SSD及以上 |
| 带宽 | 1Mbps | 5Mbps及以上 |

### 操作系统要求

| 系统类型 | 支持版本 |
|---------|---------|
| Linux | CentOS 7+/Ubuntu 18+/Debian 10+ |
| Windows | Windows Server 2012+ |
| macOS | 仅用于开发环境 |

### 软件环境要求

| 软件 | 最低版本 | 推荐版本 | 说明 |
|------|---------|---------|------|
| PHP | 7.1 | 7.4 | 必须开启PDO扩展 |
| MySQL | 5.7 | 8.0 | 推荐InnoDB引擎 |
| Redis | 3.0 | 7.0 | 可选，提升性能 |
| Nginx | 1.18 | 1.22 | 反向代理服务器 |
| Composer | 1.8 | 2.x | PHP依赖管理 |
| Git | 2.0 | 最新版本 | 代码版本控制 |

### PHP扩展要求

```
必须开启的扩展：
- fileinfo       (文件信息)
- pdo            (数据库操作)
- pdo_mysql      (MySQL驱动)
- openssl        (加密处理)
- mbstring       (字符串处理)
- curl           (HTTP请求)
- json           (JSON处理)
- session        (会话管理)

推荐开启的扩展：
- redis          (Redis缓存)
- gd             (图片处理)
- zip            (压缩处理)
- bcmath         (高精度计算)
```

## 🐳 方式一：Docker 一键部署（推荐）

### 1. 环境准备

```bash
# 检查Docker是否安装
docker --version
# Docker version 20.10.x, build xxxxx

# 检查Docker Compose
docker-compose --version
# Docker Compose version v2.x.x

# 如未安装，请先安装Docker
# Ubuntu系统安装：
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 安装Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/download/v2.20.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose
```

### 2. 获取项目代码

```bash
# 克隆项目
git clone https://gitee.com/ZhongBangKeJi/CRMEB.git crmeb
cd crmeb

# 切换到稳定版本分支（根据需要选择）
git checkout -b v5.6.4 origin/v5.6.4
```

### 3. 配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑环境变量
vim .env
```

```ini
# 数据库配置
DB_HOST=mysql
DB_PORT=3306
DB_DATABASE=crmeb
DB_USERNAME=root
DB_PASSWORD=crmeb123456

# Redis配置
REDIS_HOST=redis
REDIS_PORT=6379
REDIS_PASSWORD=

# 应用配置
APP_DEBUG=true
APP_URL=http://localhost

# Admin后台配置
ADMIN_HTTPS=false
```

### 4. 一键启动

```bash
# 构建并启动所有服务
docker-compose up -d --build

# 查看服务状态
docker-compose ps

# 输出示例：
#   Name                 Command               State           Ports
# ----------------------------------------------------------------------
# crmeb-app     docker-php-entrypoint apach ...   Up              0.0.0.0:80->80/tcp
# crmeb-mysql   docker-entrypoint.sh mysqld ...   Up              3306/tcp
# crmeb-redis   docker-entrypoint.sh redis ...   Up              6379/tcp
```

### 5. 访问安装向导

打开浏览器访问：`http://your-server-ip`

按照安装向导步骤完成系统安装：
1. 环境检测
2. 目录权限检查
3. 数据库配置
4. 管理员账号设置
5. 安装完成

### Docker 管理命令

```bash
# 启动服务
docker-compose start

# 停止服务
docker-compose stop

# 重启服务
docker-compose restart

# 查看日志
docker-compose logs -f

# 删除服务（保留数据）
docker-compose down

# 删除服务（包含数据卷）
docker-compose down -v
```

## 🖥️ 方式二：手动部署（宝塔面板）

### 1. 安装宝塔面板

```bash
# CentOS系统
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
```

### 2. 登录宝塔面板

安装完成后，记录面板登录地址、用户名和密码，登录宝塔面板。

### 3. 安装环境套件

在宝塔面板中：
1. 点击「软件商店」
2. 安装以下软件：
   - Nginx 1.22
   - MySQL 5.7/8.0
   - PHP 7.4
   - phpMyAdmin 4.7（可选，用于数据库管理）

### 4. 创建站点

1. 点击「网站」→「添加站点」
2. 填写站点信息：
   - 域名：`your-domain.com`（或IP用于测试）
   - 根目录：`/www/wwwroot/crmeb`
   - PHP版本：选择 7.4
   - 数据库：选择创建数据库
   - 数据库用户名和密码
   - FTP：选择创建

### 5. 上传项目代码

```bash

# 1. 在本地下载项目源码
```

### 6. 配置目录权限

```bash

chmod -R 755 /www/wwwroot/crmeb/runtime
chmod -R 755 /www/wwwroot/crmeb/public/uploads

# 设置所有者
chown -R www:www /www/wwwroot/crmeb
```

### 7. 配置伪静态

在宝塔面板中：
1. 点击对应站点的「设置」
2. 选择「伪静态」
3. 添加以下规则：

```nginx
# ThinkPHP 伪静态规则
location / {
    if (!-e $request_filename) {
        rewrite  ^(.*)$  /index.php?s=$1  last;
        break;
    }
}

# PHP配置
location ~ \.php$ {
    fastcgi_pass   unix:/tmp/php-cgi-74.sock;
    fastcgi_index  index.php;
    fastcgi_param  SCRIPT_FILENAME  $document_root$fastcgi_script_name;
    include        fastcgi_params;
}
```

### 8. 配置PHP上传限制

在宝塔「软件商店」→「PHP 7.4」→「配置修改」中：
- upload_max_filesize：100M
- post_max_size：100M
- max_execution_time：300
- max_input_time：300
- memory_limit：256M

打开浏览器访问：`http://your-domain.com`

### 9. 配置Nginx站点

```bash
# 创建站点配置
vim /etc/nginx/conf.d/crmeb.conf
```

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /var/www/crmeb/public;
    index index.php index.html;

    # 日志配置
    access_log /var/log/nginx/crmeb_access.log;
    error_log /var/log/nginx/crmeb_error.log;

    # 静态文件缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|woff|woff2|ttf|svg)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    # 主路由规则
    location / {
        try_files $uri $uri/ /index.php?s=$uri&$args;
    }

    # PHP处理
    location ~ \.php$ {
        fastcgi_pass 127.0.0.1:9000;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;

        # 超时配置
        fastcgi_connect_timeout 300;
        fastcgi_send_timeout 300;
        fastcgi_read_timeout 300;
    }

    # 禁止访问敏感文件
    location ~ /\.(env|git|htaccess) {
        deny all;
    }
}
```

## 🔧 安装后配置

### 1. 配置HTTPS（推荐）

```nginx
server {
    listen 443 ssl http2;
    server_name your-domain.com;

    ssl_certificate /etc/nginx/ssl/crmeb.crt;
    ssl_certificate_key /etc/nginx/ssl/crmeb.key;
    ssl_session_timeout 5m;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-RSA-AES128-GCM-SHA256:HIGH:!aNULL:!MD5:!RC4:!DHE;

    # 其他配置同上...
}

# HTTP重定向到HTTPS
server {
    listen 80;
    server_name your-domain.com;
    return 301 https://$server_name$request_uri;
}
```

### 2. 配置七牛云存储（可选）

在后台管理系统中：
1. 进入「系统设置」→「附件配置」
2. 选择「七牛云存储」
3. 配置以下参数：
   - AccessKey
   - SecretKey
   - Bucket（存储空间名称）
   - Domain（访问域名）

### 3. 配置短信服务（可选）

在后台管理系统中：
1. 进入「系统设置」→「短信设置」
2. 选择短信服务商（阿里云、腾讯云等）
3. 配置API密钥和签名
