# CRMEB 管理端前端目录结构详细说明

## 📋 项目概览

**技术栈**: Vue.js 2.x + Element UI + Vuex + SCSS  
**UI框架**: Element UI + VXE-Table + Vue Awesome Swiper  
**项目版本**: CRMEB标准版 5.6.2  
**开发模式**: 组件化 + 模块化 + 响应式布局  

## 🏗️ 整体架构设计

